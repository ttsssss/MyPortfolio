    Personal Data Spotify Web App

    A web app for visualizing personalized Spotify data. View your top artists, top tracks, and playlists.

    This app uses uses Spotify APIs, while also implementing Spotify's OAuth flow and fetching API data to create a beautiful user-friendly React app.

    *Due to Spotify's recent changes, third-party apps are automatically placed in "development mode." In development mode, users other than app developer must be explicitly added under the section "Users and Access" in the developer dashboard before they can authenticate with app. Otherwise, access cannot be granted.

    *If you would like to try app, send me your email and I can add you to "Users and Access" list!

    • Technologies: React, NodeJS, Express, Styled Components, Spotify API
